Creating initial files for a Node.js project (linux)

Create app.js and package.json on linux
 $ touch app.js package.json

Creating initial files for a Node.js project (Windows)

Create app.js and package.json on Windows
 $ type nul > app.js

Creating a package.json file

Make it easy for others to manage and install the project.
$ npm init --yes # This will trigger automatically populated initialization.

Installing Express and save it in the dependencies list.

Starting a Node.js webserver

Once Node.js is installed a webserver is started like this
$ node app.js # visit http://localhost:port and you will see your website
node

Using nodemon to start node.js apps

Nodemon is a tool that helps develop Node.js based applications
$ nodemon app.js # the Node.js app will automatically restart upon saved code changes
nodemon
